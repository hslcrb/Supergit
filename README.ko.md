# 🚀 슈퍼깃 (Supergit)

Git과 GitHub CLI 작업을 위한 사용하기 쉽고 아름다운 GUI 애플리케이션입니다.

## ✨ 기능

- 📊 **Git 상태 관리** - 스테이징된/스테이징되지 않은 변경사항 보기 및 관리
- 🌿 **브랜치 작업** - 쉽게 브랜치 생성, 체크아웃 및 확인
- 📝 **커밋 히스토리** - 상세한 정보와 함께 커밋 히스토리 탐색
- 🔀 **풀 리퀘스트 생성** - GitHub CLI를 사용한 쉬운 PR 생성 (설치된 경우)
- 🌐 **다국어 지원** - 한국어와 English
- 🎨 **아름다운 다크 테마** - 개발자를 위한 눈에 편안한 디자인

## 📋 사전 요구사항

- Node.js (v16 이상)
- npm 또는 yarn
- Git
- GitHub CLI (선택사항, PR 기능용) - [gh 설치하기](https://cli.github.com/)

## 🚀 설치

1. 이 저장소를 클론합니다:
```bash
git clone <repository-url>
cd supergit
```

2. 의존성을 설치합니다:
```bash
npm install
```

## 🏃‍♂️ 애플리케이션 실행

### 개발 모드

```bash
npm start
```

React 개발 서버와 Electron 앱이 함께 시작됩니다.

### 프로덕션 빌드

```bash
npm run build:electron
```

`dist` 폴더에 플랫폼별 배포 가능한 패키지가 생성됩니다.

## 🎯 사용법

1. **저장소 선택**: "저장소 선택"을 클릭하여 Git 저장소를 선택합니다
2. **상태 탭**: 파일 보기 및 스테이징/언스테이징, 변경사항 커밋
3. **브랜치 탭**: 새 브랜치 생성, 기존 브랜치 체크아웃
4. **커밋 탭**: 커밋 히스토리 및 상세 정보 탐색
5. **풀 리퀘스트 탭**: PR 생성 및 보기 (GitHub CLI 필요)
6. **설정 탭**: 언어 변경 및 앱 정보 확인

## 🛠️ 기술 스택

- **Electron** - 크로스 플랫폼 데스크톱 애플리케이션 프레임워크
- **React** - UI 라이브러리
- **simple-git** - Node.js의 Git 작업
- **GitHub CLI** - GitHub PR 작업
- **i18next** - 다국어 지원
